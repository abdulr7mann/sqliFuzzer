# sqliFuzzer
### A simple bash tool to fuzz for SQLi
It urlencodes payloads then uses it
```bash
./sqlfuzzing.sh url sqliPayloadList.txt
./sqlfuzzing.sh http://leettime.net/sqlninja.com/tasks/basic_ch1.php?id=1 ~/Desktop/sqlList.txt
```
